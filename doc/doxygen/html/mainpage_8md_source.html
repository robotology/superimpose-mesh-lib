9 # ‚ö†Ô∏è About versioning {#versioning}
10 
11 ---
12 The project is undergoing _heavy_ development: APIs will be subject to changes quite often.
13 To be able to understand API compatibility during development, the project will follow [SemVer](http://semver.org/) specs.
14 
15 In particular, the library will have **zero major version**, i.e. **0.MINOR.PATCH**, as specified by [SemVer spec. 4](http://semver.org/#spec-item-4) and the project will comply with the following rules:
16  1. **MINOR** version increases when API compatibility is broken;
17  2. **PATCH** version increases when functionality are added in a backwards-compatible manner;
18  3. Additional labels for pre-release and build metadata are available as extensions to the 0.MINOR.PATCH format.

21 # üìñ Background {#background}
22 
23 ---
24 This library provides superimposition facilities: the placement of one thing over another.
25 In particular, this library provides classes to superimpose 3D mesh model on top of an image that are of central importance for computer vision and augmented-reality applications.

39 # üî® Build and link the library {#build-and-link-the-library}
40 
41 ---
42 
43 Use the following commands to build, install and link the library.
44 
45 ## Build
46 
47 With `make` facilities:
48 ```bash
49 $ git clone https://github.com/robotology/superimpose-mesh-lib
50 $ cd superimpose-mesh-lib
51 $ mkdir build && cd build
52 $ cmake ..
53 $ make
54 $ [sudo] make install
55 ```
56 
57 With IDE build tool facilities:
58 ```bash
59 $ git clone https://github.com/robotology/superimpose-mesh-lib
60 $ cd superimpose-mesh-lib
61 $ mkdir build && cd build
62 $ cmake ..
63 $ cmake --build . --target ALL_BUILD --config Release
64 $ cmake --build . --target INSTALL --config Release
65 ```
66 
67 ## Link
68 Once the library is installed, you can link it using `CMake` with as little effort as writing the following line of code in your project's `CMakeLists.txt`:
69 ```cmake
70 ...
71 find_package(SuperimposeMesh 0.MINOR.PATCH EXACT REQUIRED)
72 ...
73 target_link_libraries(<target> SuperimposeMesh::SuperimposeMesh)
74 ...
75 ```
